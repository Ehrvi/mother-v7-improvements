## 🗄️ Database Knowledge Access

### **Query Examples**

```sql
-- Find all learnings about a specific phase
SELECT * FROM knowledge 
WHERE tags LIKE '%phase-15%' 
ORDER BY createdAt DESC;

-- Search for specific topics
SELECT * FROM knowledge_search 
WHERE MATCH(searchable_text) AGAINST('threshold calibration' IN NATURAL LANGUAGE MODE);

-- Get most accessed knowledge
SELECT * FROM knowledge 
ORDER BY accessCount DESC 
LIMIT 10;

-- Find learnings by category
SELECT * FROM knowledge 
WHERE category = 'Performance' 
ORDER BY createdAt DESC;

-- Get all mistakes and corrections
SELECT k.title, m.mistake, m.correction, m.prevention
FROM knowledge k
JOIN mistakes m ON k.id = m.knowledge_id
ORDER BY k.createdAt DESC;

-- Get metrics for a specific phase
SELECT k.title, me.metric_name, me.before_value, me.after_value, me.improvement_pct
FROM knowledge k
JOIN metrics me ON k.id = me.knowledge_id
WHERE k.phase = 'Phase 18'
ORDER BY me.improvement_pct DESC;
```
