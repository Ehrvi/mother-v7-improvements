## 🌊 ANIMATION PHILOSOPHY

### Timing: **Organic Easing**
```css
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
--ease-in-out-circ: cubic-bezier(0.85, 0, 0.15, 1);
--ease-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### Durations:
- Micro: 150ms (hover states)
- Short: 300ms (transitions)
- Medium: 600ms (page changes)
- Long: 1200ms (entrance animations)

### Principles:
- **Anticipation:** Elements prepare before moving
- **Follow-through:** Movement continues past target
- **Overlapping:** Stagger animations for flow
